# Fuchidori

Public pages management with Asset Pipeline.

## Usage

`rake fuchidori:generate`.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'fuchidori'
```

And then execute:

```bash
$ bundle
```

Or install it yourself as:

```bash
$ gem install fuchidori
```
